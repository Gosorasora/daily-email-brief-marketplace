# Daily Email Brief (Korean)

Gmail 받은편지함에서 **최근 24시간 이메일**을 가져와 **중요 / 참고 / 노이즈** 세 범주로 분류하고 한국어로 요약해 주는 플러그인입니다.

메인 Gmail로 **자동 전달**되는 다른 Gmail 계정이나, **POP3로 동기화**된 네이버/다음 메일도 한 번의 검색으로 모두 처리됩니다.

## 무엇을 하나요

설치하고 Gmail을 연결한 뒤 아래처럼 말하면 요약본이 나옵니다.

- "오늘 메일 요약"
- "받은편지함 정리해줘"
- "이메일 중요한거 요약"
- "daily email brief"

출력 예시:

```
## 📬 2026-04-22 이메일 요약 (오전)

총 27건 (중요 5 / 참고 15 / 노이즈 7)

### 🔴 중요/액션 필요 (5건)
- **동국대학교** — 인탑스평산장학재단 장학생 선발: 이공계 3-4학년, 연 600만원. 마감 4/24(금) 16:00. _(14:38)_
- **NHN Cloud** — 기본 인프라 비정기 점검: 4/28(화) 04:00~06:00. _(17:28)_
...

### 🟡 참고 (15건)
- **LinkedIn Job Alerts** — CJ OLIVE YOUNG 데이터 모델링 담당자 _(08:22)_
...

### 🗑️ 노이즈 7건 (광고·자동 알림 등) — 생략

---
💡 오늘 챙길 일: 4/24까지 장학금 학과 추천 문의
```

## 설치 후 할 일

1. **Gmail 커넥터 연결** — 본인 Gmail 계정으로 연결. 플러그인은 연결된 계정의 메일만 읽습니다.
2. (선택) **매일 자동 실행 스케줄 등록** — Cowork의 스케줄 기능에서 "Daily email brief" 스킬을 실행하도록 매일 아침 시간 지정. 매일 같은 포맷으로 자동 브리핑이 도착합니다.

## 분류 기준

| 범주 | 포함되는 것 |
|---|---|
| 🔴 **중요** | 사람이 직접 보낸 메일, 답장 필요한 업무 메일, 결제/주문/배송/계약/세금/금융 알림, 본인이 신청/등록한 서비스의 중요 공지, 면접/일정 조율, **기한이 명시된 메일** |
| 🟡 **참고** | 채용 공고(LinkedIn 자동 추천 포함), 본인이 구독한 뉴스레터 중 본문 가치가 있는 것, 본인 행동이 아닌 시스템/보안 알림 |
| 🗑️ **노이즈** | 제목에 `[AD]` `광고` `프로모션` `Sale` `Deal` 포함, 단순 마케팅, 본인이 방금 한 행동에 대한 시스템 알림(2FA, 인증번호 등) |

노이즈는 요약에 포함하지 않고 **건수만** 표시합니다.

## 다른 언어로 쓰고 싶다면

기본 출력이 한국어지만, "영어로 요약해줘" 같이 다른 언어를 지정하면 그 언어로 생성합니다.

## 요구사항

- Gmail 커넥터 (`search_threads`, `get_thread` 툴 제공)
