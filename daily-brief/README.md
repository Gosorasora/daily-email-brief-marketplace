# 데일리브리프 (daily-brief)

한국어 데일리 브리핑 스킬 묶음. 모두 **읽기 전용 리포트**만 생성하며, 읽음/스팸/발송/캘린더 자동 등록 같은 쓰기 작업은 하지 않습니다.

## 포함 스킬
| 스킬 | 하는 일 | 트리거 예시 |
|---|---|---|
| `daily-brief` | Gmail(24h) + Slack(7d) 통합 브리핑 | "데일리 브리핑", "오늘 브리핑" |
| `email-brief` | Gmail 받은편지함 24시간 요약 | "이메일 요약", "메일 브리핑" |
| `slack-brief` | Slack 채널 최근 7일 요약 | "슬랙 요약", "채널 요약" |

## 설치 후 해야 할 일 (중요)
스킬은 그대로 따라오지만, 아래는 **각자 직접** 설정해야 동작합니다.
1. **커넥터 연결**: Gmail, Slack (캘린더는 선택). 본인 계정으로 인증. — `CONNECTORS.md` 참고.
2. **설정값 입력**: 각 SKILL.md 상단 `⚙️ 설정` 블록의 `<...>` 자리표시자(메일 주소, 슬랙 워크스페이스/채널 ID)를 본인 값으로 교체.
3. (선택) **스케줄 등록**: 매일 정해진 시각 자동 실행을 원하면 schedule 스킬로 등록.

## 참고
- Gmail 외부계정 가져오기(POP3)는 2026년 종료되었습니다. 네이버 등 외부 메일을 함께 보려면 해당 계정에서 Gmail로 **자동전달**을 설정하세요.
- 개인 메일·슬랙 데이터는 이 플러그인에 포함되어 있지 않습니다.
